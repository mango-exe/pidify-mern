List of key components
e following is a summary of key components in the extracted JSON:
Elements: Ordered list of semantic elements (like headings, paragraphs, tables, gures) found in the
document, on the basis of position in the structure tree of the document. e output does not include
headers or footers. In addition, headings that repeat across pages are reported for the rst occurrence
only.
Bounds: Bounding box enclosing the content items forming this element. Not reported for elements
which dont have any content items (like empty table cells).
Font: Font description for the font associated with the rst character. Only reported for text elements.
TextSize: Text size (in points) of the last character. Only reported for text elements.
Aributes: Includes additional properties like line height and text alignment.
Path: e Path describes the location of elements in the structure tree including the element type and
the instance number. Element types are based on the ISO standard
